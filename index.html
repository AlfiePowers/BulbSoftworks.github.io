Who Am I?

My name is Alfie Powers, and I am a prospecting Computer Science student. My main projects consist of my Discord Bot, which the maintanence of, takes a fair amount of my time. If you are intrested in any of my code, please check out my source code for the bot, and if there are any issues you find, either PM me on GitHub or attempt to fix them yourself by forking the repository. Any bug fixes are gladly accepted, and I will make sure that I add you to the credits on the GitHub page.

If you're intrested in either my project or the Java Discord API (JDA), I thouroughly reccomend reading the documentation and/or dv8tion's source code. Thanks to dv8tion for the amazing API that is JDA as without it, Guardian wouldn't exist.
